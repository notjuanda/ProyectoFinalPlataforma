Este curso te introduce al uso de CSS para diseñar y estilar páginas web, cubriendo desde los fundamentos hasta técnicas avanzadas de diseño.

Aprende a utilizar las poderosas herramientas de CSS Flexbox y Grid Layout para crear diseños web complejos y adaptables.

Comprende los conceptos básicos de JavaScript, el lenguaje de programación que da vida a las páginas web, con ejemplos prácticos y ejercicios.

Descubre cómo manejar operaciones asíncronas en JavaScript usando promesas, async/await y otros patrones importantes.

APIs y AJAX

Aprende a interactuar con APIs web y a utilizar AJAX para crear experiencias de usuario dinámicas y sin interrupciones.
